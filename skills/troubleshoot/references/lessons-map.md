1. Read each applicable file
2. Search for keywords from the problem (e.g., "widget", "container", "routing", "hydration")
3. List every lesson that matches or is related
4. Check if any lesson directly solves the problem
5. If a lesson says "always do X first" — do X first, even if you think you already know the answer
